# Requisitos Funcionais - Sistema Rede de Cinemas

## RF01 – Cadastro de Cinemas
O sistema deve permitir cadastrar cinemas com:
- Nome
- Cidade
- Estado
- Endereço
- Capacidade de público

## RF02 – Cadastro de Filmes
O sistema deve permitir cadastrar filmes com:
- Título
- Duração (em minutos)
- Gênero
- Diretor
- Elenco

## RF03 – Cadastro de Sessões
O sistema deve permitir cadastrar sessões vinculadas a:
- Um cinema específico
- Um filme específico
- Data e horário de início

## RF04 – Registro de Público
O sistema deve permitir registrar a quantidade de espectadores em cada sessão

## RF05 – Consulta de Filmes em Cartaz
O sistema deve permitir consultar os filmes em exibição:
- Filmes por cinema
- Sessões disponíveis por filme

## RF06 – Relatórios
O sistema deve permitir consultar:
- Público por sessão
- Público total por filme
- Público total por cinema
- Ocupação percentual por sessão
