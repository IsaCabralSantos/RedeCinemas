# Requisitos Funcionais - Sistema Rede de Cinemas

## RF01 – Cadastro de Cinemas
O sistema deve permitir cadastrar cinemas com:
- Nome
- Cidade
- Estado
- Endereço
- Capacidade de público
<img width="352" height="404" alt="image" src="https://github.com/user-attachments/assets/a4fd0e3c-ec6c-4901-91cc-5458206248c8" />

## RF02 – Cadastro de Filmes
O sistema deve permitir cadastrar filmes com:
- Título
- Duração (em minutos)
- Gênero
- Diretor
- Elenco
<img width="319" height="404" alt="image" src="https://github.com/user-attachments/assets/003ea796-d440-433b-861b-f924cb05d78d" />

## RF03 – Cadastro de Sessões
O sistema deve permitir cadastrar sessões vinculadas a:
- Um cinema específico
- Um filme específico
- Data e horário de início
<img width="484" height="427" alt="image" src="https://github.com/user-attachments/assets/074c1f18-289b-4049-8c35-4176ac205903" />

## RF04 – Registro de Público
O sistema deve permitir registrar a quantidade de espectadores em cada sessão
<img width="432" height="306" alt="image" src="https://github.com/user-attachments/assets/e17be89b-75bc-4b9f-8e68-2999658e2325" />

## RF05 – Consulta de Filmes em Cartaz
O sistema deve permitir consultar os filmes em exibição:
- Filmes por cinema
- Sessões disponíveis por filme
<img width="360" height="306" alt="image" src="https://github.com/user-attachments/assets/2b7b184b-c551-42ba-b94d-065751a9f204" />


## RF06 – Relatórios
O sistema deve permitir consultar:
- Público por sessão
- Público total por filme
- Público total por cinema
- Ocupação percentual por sessão
